### DBの一覧を表示する

```sql
SHOW DATABASES;
```

---

### 使用するDBを選択する

```sql
USE <database_name>;
```

---

### テーブル一覧を表示する

```sql
SHOW TABLES;
```

---

### テーブル定義を見る

```sql
DESC <table_name>;
```

---

### 指定したテーブルのCREATE TABLE文を見る

```sql
SHOW CREATE TABLE <table_name>;
```
